# 🚌 Bus Reservation System (WebAssembly)

A fully interactive **Bus Reservation System** built in **C++** and deployed as a web application using **WebAssembly (Emscripten)** with a modern terminal-style UI.

## 📌 Features

* 📍 View available routes
* 🎟️ Book tickets with seat selection
* ❌ Cancel tickets
* 🔍 Search ticket details
* 💻 Interactive terminal-style interface
* 🌐 Runs completely in the browser (no backend required)

---

## 🧠 Tech Stack

* **C++** – Core logic & data structures
* **WebAssembly (WASM)** – Run C++ in browser
* **Emscripten** – Compilation toolchain
* **HTML + CSS + JavaScript** – Frontend UI

## ⚙️ How It Works

```
User Input → JavaScript → stdin → WebAssembly (C++)
C++ Output → print() → Browser UI
```

## 🛠️ Setup & Run Locally

### 1️⃣ Install Emscripten

```bash
git clone https://github.com/emscripten-core/emsdk.git
cd emsdk
emsdk install latest
emsdk activate latest
```

---

### 2️⃣ Activate Environment

#### Windows (PowerShell):

```powershell
cd D:\emsdk
.\emsdk_env.ps1
```

---

### 3️⃣ Compile C++ to WebAssembly

```bash
emcc Main.cpp -o index.html -sASYNCIFY -sFORCE_FILESYSTEM -sEXIT_RUNTIME=1 -sASSERTIONS=0
```

---

### 4️⃣ Replace Generated HTML

Delete the generated `index.html` and use your custom UI file.

---

### 5️⃣ Run Locally

Open `index.html` in a browser
(or use Live Server in VS Code)

## ⚠️ Important Notes

* Do **not** use `cin` → replaced with `scanf` for WASM compatibility
* Use `.at()` instead of `map[key]` to avoid constructor errors
* Ensure `stdin` in JavaScript does NOT return `null` (prevents auto-exit)
* Always compile with:

  ```
  -sEXIT_RUNTIME=1
  ```
